# Just Metal Shapes

`Just Metal Shapes` is a concept that blends the chaotic rhythm-bullet-hell feel of **Just Shapes & Beats** with the cooperative action identity of **Full Metal Furies**.

## Core Idea

The base gameplay loop is intentionally close to **Just Shapes & Beats**:

- Fast movement and dodging through dense attack patterns
- Tight, reactive control
- Cooperative survival-focused play

On top of that foundation, this project adds combat-focused twists inspired by team-based action games.

## Gameplay Twists

### 1) Color-Based Enemy Shields

Enemies can spawn with shields tied to player color.  
This means players may need the matching color state (or teammate coordination) to efficiently break shields and create openings.

### 2) Proximity-Based Damage Rules

Certain enemies can only be damaged when players are within specific range thresholds.  
This adds positional decision-making on top of dodging: sometimes you must get close to deal damage, while still avoiding incoming patterns.

## Design Goal

Create a game that keeps the kinetic readability and musical pressure of JS&B, while adding tactical co-op interactions through shield matching and distance-based damage windows.
